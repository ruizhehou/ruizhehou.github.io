智能体编程深度解析

一、智能体编程概述

智能体编程（Agentic Programming）是随着大语言模型（LLM）发展而兴起的一种编程范式。它将AI模型视为能够自主思考、规划、执行任务的"智能体"，通过构建多智能体协作系统来解决复杂问题。

1.1 什么是智能体

智能体（Agent）是一个能够感知环境、做出决策并执行动作的自主实体。在AI领域，智能体通常基于大语言模型，具备以下核心能力：

1.2 智能体编程的特点

特点 说明 传统编程对比
自主性 智能体可以自主决策 需要人工编写详细逻辑
适应性 能根据环境变化调整 固定的程序逻辑
推理能力 可以理解和推理 只能执行预设指令
工具使用 可以调用各种工具 需要手动集成
多智能体协作 多个智能体协同工作 单一程序

1.3 应用场景

  • 代码生成与重构
  • 自动化测试
  • 数据分析与报告
  • 客户服务与支持
  • 研究与知识发现
  • 创意内容生成
  • 决策支持系统

2.3 ReAct模式

ReAct（Reasoning + Acting）是最常用的智能体推理模式。

Thought 1: 用户想要搜索信息
Action 1: 调用搜索工具
Observation 1: 获得搜索结果
Thought 2: 分析搜索结果，提取关键信息
Action 2: 调用分析工具
Observation 2: 获得分析结果
Thought 3: 综合信息，生成最终回答
Action 3: 返回结果给用户

3.3 多智能体协作系统

from typing import List, Dict, Any
from dataclasses import dataclass
from langchain.chat_models import ChatOpenAI
from langchain.prompts import ChatPromptTemplate

@dataclass
class Message:
    role: str
    content: str

class Agent:
    def __init__(self, name: str, role: str, llm):
        self.name = name
        self.role = role
        self.llm = llm
        self.memory: List[Message] = []

    def think(self, context: str) -> str:
        """思考并生成响应"""
        prompt = ChatPromptTemplate.from_messages([
            ("system", f"你是{self.name}，你的角色是：{self.role}"),
            ("human", "上下文：{context}\n\n请基于你的角色给出建议或行动。")
        ])

        messages = prompt.format_messages(context=context)
        response = self.llm(messages).content

        # 记录到记忆
        self.memory.append(Message(role="assistant", content=response))
        return response

class Coordinator:
    def __init__(self):
        self.agents: Dict[str, Agent] = {}
        self.shared_memory: List[Message] = []

    def register_agent(self, name: str, role: str, llm) -> Agent:
        """注册智能体"""
        agent = Agent(name, role, llm)
        self.agents[name] = agent
        return agent

    def coordinate(self, task: str, max_rounds: int = 3) -> str:
        """协调多智能体协作完成任务"""
        context = f"任务：{task}\n\n"

        for round_num in range(max_rounds):
            print(f"\n=== 第{round_num + 1}轮协作 ===")

            # 让每个智能体轮流发言
            for agent_name, agent in self.agents.items():
                response = agent.think(context)
                print(f"{agent_name}: {response}")

                # 更新共享上下文
                context += f"\n{agent_name}: {response}"

                # 记录到共享记忆
                self.shared_memory.append(
                    Message(role=agent_name, content=response)
                )

            # 检查是否达成共识
            if self.check_consensus():
                break

        return context

    def check_consensus(self) -> bool:
        """检查智能体是否达成共识"""
        # 简化实现：如果最近3条消息相似，认为达成共识
        recent = self.shared_memory[-3:]
        if len(recent) < 3:
            return False

        # 实际应用中应该使用更复杂的共识检测逻辑
        return True

# 使用示例
def main():
    # 初始化LLM
    llm = ChatOpenAI(temperature=0.7)

    # 创建协调器
    coordinator = Coordinator()

    # 注册多个智能体
    coordinator.register_agent("产品经理", "负责需求分析和产品规划", llm)
    coordinator.register_agent("架构师", "负责技术架构设计", llm)
    coordinator.register_agent("开发工程师", "负责代码实现", llm)
    coordinator.register_agent("测试工程师", "负责质量保证", llm)

    # 执行任务
    task = "设计一个在线文档协作系统的技术方案"
    result = coordinator.coordinate(task, max_rounds=3)

    print("\n=== 最终方案 ===")
    print(result)

if __name__ == "__main__":
    main()

3.4 AutoGPT风格的自规划智能体

from typing import List, Optional
from dataclasses import dataclass
import json
from langchain.chat_models import ChatOpenAI
from langchain.prompts import ChatPromptTemplate

@dataclass
class Task:
    id: int
    description: str
    status: str = "pending"  # pending, in_progress, completed, failed
    result: Optional[str] = None

class AutoAgent:
    def __init__(self, name: str, llm):
        self.name = name
        self.llm = llm
        self.tasks: List[Task] = []
        self.memory: List[dict] = []
        self.task_counter = 0

    def plan(self, goal: str) -> List[Task]:
        """将目标分解为任务列表"""
        prompt = ChatPromptTemplate.from_messages([
            ("system", "你是一个任务规划专家。"),
            ("human", """
目标：{goal}

请将这个目标分解为3-5个具体的、可执行的子任务。
每个任务应该清晰、具体、可验证。

以JSON格式返回，格式如下：
{{
    "tasks": [
        {{"description": "任务1描述"}},
        {{"description": "任务2描述"}},
        ...
    ]
}}
""")
        ])

        messages = prompt.format_messages(goal=goal)
        response = self.llm(messages).content

        # 解析JSON响应
        try:
            data = json.loads(response)
            tasks = []
            for task_data in data["tasks"]:
                self.task_counter += 1
                task = Task(
                    id=self.task_counter,
                    description=task_data["description"]
                )
                tasks.append(task)
                self.tasks.append(task)
            return tasks
        except json.JSONDecodeError:
            print("解析任务失败，使用默认规划")
            return [Task(id=1, description=goal)]

    def execute_task(self, task: Task) -> str:
        """执行单个任务"""
        task.status = "in_progress"

        prompt = ChatPromptTemplate.from_messages([
            ("system", "你是一个任务执行专家。"),
            ("human", """
任务：{task_description}

请执行这个任务并提供详细的结果。
如果任务需要调用工具或API，请模拟调用过程。
""")
        ])

        messages = prompt.format_messages(task_description=task.description)
        response = self.llm(messages).content

        task.result = response
        task.status = "completed"

        # 记录到记忆
        self.memory.append({
            "task_id": task.id,
            "description": task.description,
            "result": response
        })

        return response

    def reflect(self) -> str:
        """反思执行过程，生成总结"""
        if not self.memory:
            return "还没有执行任何任务。"

        prompt = ChatPromptTemplate.from_messages([
            ("system", "你是一个反思专家。"),
            ("human", """
以下是已执行的任务历史：
{task_history}

请分析这些任务的执行情况，总结：
1. 完成了什么
2. 遇到了什么问题
3. 有什么改进建议
""")
        ])

        task_history = "\n".join([
            f"任务{m['task_id']}: {m['description']}\n结果: {m['result']}\n"
            for m in self.memory
        ])

        messages = prompt.format_messages(task_history=task_history)
        reflection = self.llm(messages).content

        return reflection

    def run(self, goal: str) -> str:
        """运行智能体，完成目标"""
        print(f"🎯 目标：{goal}\n")

        # 1. 规划
        print("📋 规划阶段...")
        tasks = self.plan(goal)
        print(f"✅ 生成了{len(tasks)}个子任务\n")

        # 2. 执行
        print("⚙️ 执行阶段...")
        for task in tasks:
            print(f"\n🔧 执行任务{task.id}: {task.description}")
            result = self.execute_task(task)
            print(f"✅ 任务{task.id}完成")

        # 3. 反思
        print("\n💭 反思阶段...")
        reflection = self.reflect()
        print(reflection)

        return reflection

# 使用示例
def main():
    llm = ChatOpenAI(temperature=0.7)
    agent = AutoAgent("AutoGPT", llm)

    goal = "创建一个简单的博客系统"
    agent.run(goal)

if __name__ == "__main__":
    main()

3.5 具有工具使用能力的智能体

from typing import Callable, Dict, Any
from langchain.chat_models import ChatOpenAI
from langchain.prompts import ChatPromptTemplate
import json

class Tool:
    def __init__(self, name: str, description: str, func: Callable):
        self.name = name
        self.description = description
        self.func = func

    def run(self, **kwargs) -> Any:
        return self.func(**kwargs)

class ToolUsingAgent:
    def __init__(self, name: str, llm):
        self.name = name
        self.llm = llm
        self.tools: Dict[str, Tool] = {}
        self.memory = []

    def register_tool(self, tool: Tool):
        """注册工具"""
        self.tools[tool.name] = tool

    def get_tool_schema(self) -> str:
        """获取工具的JSON Schema描述"""
        schemas = []
        for tool in self.tools.values():
            schema = {
                "name": tool.name,
                "description": tool.description
            }
            schemas.append(schema)
        return json.dumps(schemas, ensure_ascii=False, indent=2)

    def decide_action(self, user_input: str) -> Dict[str, Any]:
        """决定使用哪个工具以及如何调用"""
        prompt = ChatPromptTemplate.from_messages([
            ("system", """
你是一个工具使用专家。根据用户输入，决定是否需要调用工具。

可用工具：
{tool_schema}

如果需要调用工具，返回JSON格式：
{{
    "action": "tool_name",
    "parameters": {{"param1": "value1", ...}},
    "reasoning": "选择这个工具的原因"
}}

如果不需要调用工具，返回：
{{
    "action": "none",
    "response": "直接回答用户"
}}
"""),
            ("human", "用户输入：{user_input}")
        ])

        messages = prompt.format_messages(
            tool_schema=self.get_tool_schema(),
            user_input=user_input
        )

        response = self.llm(messages).content

        try:
            return json.loads(response)
        except json.JSONDecodeError:
            return {"action": "none", "response": response}

    def run(self, user_input: str) -> str:
        """运行智能体"""
        # 记录用户输入
        self.memory.append({"role": "user", "content": user_input})

        # 决定行动
        decision = self.decide_action(user_input)

        if decision["action"] == "none":
            # 直接回答
            response = decision["response"]
        else:
            # 调用工具
            tool_name = decision["action"]
            parameters = decision["parameters"]

            print(f"🔧 调用工具: {tool_name}")
            print(f"📋 参数: {parameters}")

            tool = self.tools.get(tool_name)
            if tool:
                result = tool.run(**parameters)
                response = f"工具执行结果：{result}"
            else:
                response = f"错误：找不到工具 {tool_name}"

        # 记录响应
        self.memory.append({"role": "assistant", "content": response})

        return response

# 定义工具示例
def calculator(operation: str, a: float, b: float) -> float:
    """计算器工具"""
    if operation == "add":
        return a + b
    elif operation == "subtract":
        return a - b
    elif operation == "multiply":
        return a * b
    elif operation == "divide":
        return a / b if b != 0 else "错误：除数不能为零"
    else:
        return "错误：不支持的操作"

def weather_query(city: str) -> str:
    """天气查询工具（模拟）"""
    # 实际应用中应该调用真实的天气API
    weather_data = {
        "北京": "晴，温度25°C",
        "上海": "多云，温度22°C",
        "深圳": "阴，温度28°C"
    }
    return weather_data.get(city, "未找到该城市的天气信息")

# 使用示例
def main():
    llm = ChatOpenAI(temperature=0)
    agent = ToolUsingAgent("工具智能体", llm)

    # 注册工具
    agent.register_tool(Tool(
        name="calculator",
        description="执行数学计算，支持加减乘除",
        func=calculator
    ))

    agent.register_tool(Tool(
        name="weather_query",
        description="查询城市天气",
        func=weather_query
    ))

    # 测试
    queries = [
        "123 + 456等于多少？",
        "今天北京的天气怎么样？",
        "计算100除以5的结果"
    ]

    for query in queries:
        print(f"\n用户: {query}")
        response = agent.run(query)
        print(f"智能体: {response}\n")

if __name__ == "__main__":
    main()

四、优缺点分析

4.1 优点

4.1.1 灵活性强

传统编程：固定逻辑
┌─────────────────────────┐

4.2 缺点

4.2.1 不可预测性

传统程序：确定性输出
输入A → 程序 → 输出A（总是相同）

智能体：概率性输出
输入A → 智能体 → 输出A（可能不同）
输入A → 智能体 → 输出B（可能不同）
输入A → 智能体 → 输出C（可能不同）

4.2.2 成本高昂

成本类型 说明 影响
API调用成本 每次推理都需要调用LLM API 按使用量计费
延迟成本 多轮推理增加响应时间 用户体验下降
资源消耗 需要维护多个智能体实例 基础设施成本

4.2.4 幻觉问题

智能体可能会产生不真实的信息：

用户：量子计算机的原理是什么？

智能体（幻觉）：
量子计算机使用量子比特，每个量子比特可以同时处于0和1的状态，
通过量子纠缠实现超光速通信，可以瞬间解决所有数学问题...

实际情况：
量子计算机确实利用量子叠加和纠缠，但不能超光速通信，
也不是万能的，只能解决特定类型的问题。

4.3 对比总结

class MultimodalAgent:
    """多模态智能体"""

    def process(self, input_data):
        """处理多模态输入"""
        # 文本理解
        if "text" in input_data:
            text_understanding = self.understand_text(input_data["text"])

        # 图像理解
        if "image" in input_data:
            image_understanding = self.understand_image(input_data["image"])

        # 音频理解
        if "audio" in input_data:
            audio_understanding = self.understand_audio(input_data["audio"])

        # 跨模态推理
        result = self.cross_modal_reasoning(
            text_understanding,
            image_understanding,
            audio_understanding
        )

        return result

5.4 自我进化智能体

class SelfEvolvingAgent:
    """自我进化智能体"""

    def __init__(self):
        self.version = 1
        self.performance_metrics = []
        self.improvement_history = []

    def evolve(self):
        """自我进化"""
        # 1. 分析当前性能
        current_performance = self.evaluate_performance()

        # 2. 识别改进点
        improvement_areas = self.identify_improvements(current_performance)

        # 3. 生成改进方案
        improvement_plan = self.generate_improvement_plan(improvement_areas)

        # 4. 应用改进
        self.apply_improvements(improvement_plan)

        # 5. 验证改进效果
        new_performance = self.evaluate_performance()

        # 6. 如果改进有效，保留；否则回滚
        if new_performance > current_performance:
            self.version += 1
            self.improvement_history.append({
                "version": self.version,
                "improvement": improvement_plan,
                "performance": new_performance
            })
        else:
            self.rollback_improvements(improvement_plan)

"""

        if intent == "schedule":
            return self.handle_schedule(user_request)
        elif intent == "task":
            return self.handle_task(user_request)
        elif intent == "search":
            return self.handle_search(user_request)
        elif intent == "advice":
            return self.handle_advice(user_request)

项目2：代码审查智能体

"""
目标：构建一个能够自动审查代码的智能体

功能需求：
1. 代码质量检查
2. 安全漏洞检测
3. 性能优化建议
4. 最佳实践推荐
"""

class CodeReviewAgent:
    def __init__(self):
        self.linter = LinterTool()
        self.security_scanner = SecurityTool()
        self.performance_analyzer = PerformanceTool()

    def review(self, code):
        """审查代码"""
        # 1. 代码质量检查
        quality_issues = self.linter.check(code)

        # 2. 安全检查
        security_issues = self.security_scanner.scan(code)

        # 3. 性能分析
        performance_issues = self.performance_analyzer.analyze(code)

        # 4. 生成审查报告
        report = self.generate_report({
            "quality": quality_issues,
            "security": security_issues,
            "performance": performance_issues
        })

        return report

项目3：研究助手智能体

"""
目标：构建一个能够辅助学术研究的智能体

功能需求：
1. 文献搜索
2. 论文总结
3. 研究方向建议
4. 写作辅助
"""

class ResearchAssistant:
    def __init__(self):
        self.paper_search = PaperSearchTool()
        self.summarizer = SummarizerTool()
        self.citation_manager = CitationTool()

    def assist_research(self, topic):
        """辅助研究"""
        # 1. 搜索相关论文
        papers = self.paper_search.search(topic)

        # 2. 总结论文内容
        summaries = []
        for paper in papers:
            summary = self.summarizer.summarize(paper)
            summaries.append(summary)

        # 3. 分析研究方向
        research_directions = self.analyze_directions(summaries)

        # 4. 生成研究建议
        suggestions = self.generate_suggestions(research_directions)

        return {
            "papers": papers,
            "summaries": summaries,
            "directions": research_directions,
            "suggestions": suggestions
        }

  1. 理解智能体：智能体具备感知、推理、行动、记忆四大核心能力

  3. 实践应用：通过项目实践积累经验
  4. 持续学习：跟上快速发展的技术节奏
